## Backend Setup:

1. Ensure you install `poetry` version `1.8.3`. Run `pip install --user poetry==1.8.3`.
2. Add the following to your `~/.zshrc` by running `nano ~/.zshrc` and then adding `export PATH="$HOME/.local/bin:$PATH"` to the file.
3. Run `source ~/.zshrc`.
3. `cd app/backend`
4. `poetry install`
5. `poetry shell`
